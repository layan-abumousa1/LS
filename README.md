# Envelope Luxury Invitation — React

A responsive one-page wedding invitation built with React and CSS animations.

## Run

```bash
npm.cmd install
npm.cmd run dev
```

Open the local URL Vite prints in the terminal, usually:

```txt
http://localhost:5173
```

## Edit content

Open `src/main.jsx` and edit the `invitation` object at the top.

## Replace photos

Replace these files inside `src/assets/`:

- `couple-main.jpg`
- `couple-soft.jpg`
- `bg.jpg`

Keep the same file names and the site updates automatically.
